# AWS Static Website Hosting

## 🔗 Live URL
**https://d1uayiedgqbob6.cloudfront.net**

## 📌 Project Overview
Deployed a fully functional static website on AWS using
S3 for storage and CloudFront as a CDN for fast,
secure, and globally available content delivery.

## AWS Services Used
- Amazon S3 — File storage (Mumbai region)
- Amazon CloudFront — Global CDN + HTTPS

## Architecture
User → CloudFront (CDN) → S3 Bucket (Mumbai)

## Steps I Followed
1. Created AWS Free Account
2. Created S3 bucket in Mumbai region
3. Uploaded website files to S3
4. Enabled Static Website Hosting on S3
5. Created CloudFront distribution
6. Website is live with HTTPS

## What I Learned
- How to host a website on AWS S3
- How CloudFront CDN works
- How to secure website with HTTPS
- AWS Free Tier usage
